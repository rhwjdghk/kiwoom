자가용

올림픽도로

영등포R → 영등포역앞(경인국도46번) → 신도림역 → 고척교 → 동양미래대학교 골목길 유턴→ 고척 스카이돔

강변북로

마포대교 → 여의도광장앞 → 서울교 → 영등포역앞 → 고척교 → 동양미래대학교 골목길 유턴→ 고척 스카이돔

남부순환로

오류IC → 고척교방향 → 구로소방서 → 고척스카이돔

서부간선(서서울IC)

고척교에서 OUT → 롯데마트 좌회전 → 고척교 → 동양미래대학교 골목길 유턴 → 고척 스카이돔

서부간선(성산대교)

고척교에서 OUT → 롯데마트 우회전 → 고척교 → 동양미래대학교 골목길 유턴 → 고척 스카이돔

경부고속도로

올림픽도로 → 김포공항방향 → 영등포R → 고척교 → 동양미래대학교 골목길 유턴 → 고척 스카이돔

영동고속도로

신갈분기점 → 안산분기점JC(성산대교방향) → 서서울TG → 고척교OUT → 롯데마트 좌회전 → 고척 스카이돔

지하철

1 2 신도림역

1번, 5번 출구에서 버스 이용 시 3정거장 소요
버스 : 160번 외 9개 노선운행

2구로역

2번 출구에서 좌회전하여 1km 직진 후, 고척교 건너 좌측 위치(도보 약 20분)
버스: 5712번, 5626번

1개봉역

2번 출구에서 200m 직진 후, 우회전하여 650m 직진 후 우측 위치(도보 약 15분)
버스: 마을버스06번 (고척교 방향)

1구일역

2번 출구에서 약 100m 직진 후 고척스카이돔 정면 출입문 게이트 통해 입장 (도보 약 3분)
버스 이용 필요 없음
